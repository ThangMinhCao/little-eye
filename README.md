# Hackathon App

## 1. Requirements
- `Python3`
- `pip`
   
## 2. Install
```
    python3 -m venv venv
    pip install -r requirements.txt
```

## 3. Start server

  ```
    FLASK_APP=app.py flask run --host=0.0.0.0
  ```
    
## 4. Test

  Send requests to a route of the host:
  ```
    http://localhost:5000/
  ```
